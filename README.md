# 1. Descrição do dataset
O dataset em questão consiste no conjunto de dados dos veiculos que foram registrados em janeiro de 2026, o dataset foi pego diretamente do Kaggle. Possuindo 45541 instancias e 14 atributos analisados

# 2. Justificativa da escolha
Este dataset é adequado para o uso do projeto, pois, além de possuir diversos erros, como diversos valores nulos.Além de consistir em dados reais com uma complexidade alta, contendo mais de 45000 instancias para serem analisadas, além de possuir 14 atributos que variam entre dados qualitativos e quantitativos, permitindo a criação de gráficos e informações mais ricas e dinâmicas.
